# Sorting Algorithms in C

This repository contains implementations of **21 sorting algorithms** in C. It is intended for educational purposes, providing examples of various sorting methods, from simple algorithms like Bubble Sort to more complex ones like Tim Sort and Bitonic Sort.

## Algorithms Included
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort
- Counting Sort
- Radix Sort
- Bucket Sort
- Shell Sort
- Tim Sort
- Comb Sort
- Pigeonhole Sort
- Cycle Sort
- Bitonic Sort
- Gnome Sort
- Odd-Even Sort
- Strand Sort
- Patience Sort
- Bogo Sort
- Stooge Sort

## How to Use
Clone this repository and compile the source files:

```bash
git clone https://github.com/yourusername/sorting-algorithms-c.git
cd sorting-algorithms-c
gcc sorting_algorithms.c -o sorting_algorithms
./sorting_algorithms
